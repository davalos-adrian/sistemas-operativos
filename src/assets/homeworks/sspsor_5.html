Herramientas para administración de dominios

Existen numerosas herramientas que nos pueden ayudar a gestionar un active directory en windows server. Para acceder a estas solamente tenemos que acceder a Inicio > Herramientas administrativas. En esta ocasion solamente hablaremos de las herramientas más importantes, porque como se puede observar, son muchas.

Administración de equipos

Con esta herramienta podemos adminsitrar y configurar opciones basicas del controlador de dominio, como administrar usuarios y grupos locales, espacio de almacenamiento en disco, recursos compartidos, etc. Esta herramienta es una herramienta de gran tamaño, ya que posee muchas características, de las cuáles se hablará en seguida.

Herramientas del sistema
  • Programador de tareas: Su funcionamiento es practicamente el mismo que en versiones normales de Windows.
  • Visor de tareas: En esta opción, se puede analizar un histórico de sucesos que han ocurrido en el sistema, ya sean errores de Inicio o cierres del equipo, interrupción de servicios, etc.
  • Carpetas compartidas: Nos muestra en una sola ventrana todos los recursos compartidos desde este equipo.
  • Rendimiento: Podemos analizar los recursos consumidos por el equipo en cuanto a CPU, disco, red y memoria.
Almacenamiento

En esta opción se puede gestionar discors duros o volumenes en el sistema de forma similar a versiones normales de windows.

Servicios y aplicaciones

Aqui se gestionan servicios y aplicaciones, aquellos que se estñan ejecutando en el equipo.

DNS

Desde aquí se puede administrar el servicios de DNS, que permite asociar nombres de host con direcciones IP. Durante el proceso de instalación de active directory se instala y se configura automáticamente (como se mostró en la tarea pasada)

Dominios y confianzas de Active Directory

Esta herramienta permite configurar algo muy importante, las relaciones de confianza. Estas se configuran cuando en una organizacion existen dos o más dominios diferentes y se pueden unificar la administración de toda la organización desde cualquier controlador de dominio (existente)

Usuario y equipos de Active Directory

Desde esta herramienta podemos administrar usuarios globales, grupos y equipos del dominio, entre otras cosas. Esta herramienta es como la de administración de equipos, pero se utiliza exclusivamente para objetos del dominio.

Conclusión

Estas herramientas nos permiten controlar los dominios de manera eficiente, puesto que existen multiples herramientas con distintas aplicaciones para controlar usuarios, grupos e incluso gracias a la herramienta de confianzas de active directory podemos unificar varios controladores cuando se encuentran varios activos dentro del dominio.
