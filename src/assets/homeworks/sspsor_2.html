Planificación de instalación de un SOR

Cuando se planea instalar un sistema operativo en red, hay que tener en cuenta muchos aspectos para poder proporcionar el servicio que se planea dar. Algunos de estos aspectos los podemos enmarcar a continuación.

  • Hardware
  • Particionamiento del disco o de los discos duros
  • Sistema de archivos a utilizar
  • Configuación de red
  • Sistema de control o autentificación para usuarios
  • Sistema operativo

Estos son algunos de los aspectos más importantes a la hora de realizar la instalación, pero existen otros que se tratan una vez terminada esta fase, por lo tanto no entran en ésta planificación.

Particiones

Cuando se tenga realizada la planificación y decidida la forma de gestionar nuestro sistema en la red, es cuando se debe empezar el proceso de instalación. Una partición es el nombre genérico que recibe cada división presente en una sola unidad física de almacenamiento de datos.

Éstas particiones las reconoce el sistema operativo y se le puede asignar un sistema de archivos a cada partición. Se presentan distintas ventajas cuando se particiona un disco duro, como las siguientes:

  • Instalar varios sistemas operativos
  • Mejorar el rendimiento
  • Mejor mantenimiento
  • Mayor facilidad en la organización

Los sistemas operativos cuentan con gestionadores de particiones para facilitar el proceso, incluso algunos cuentan con interfaces gráficas para hacerlo de manera rápida, e intuitiva

Sistema de archivos

Se puede definir un sistema de archivos como un conjunto de normas y procesos que se utilizan para almacenar la información. Se le denomina “sistema de archivos” porque cada archivo contiene unos datos independientes, por lo que es necesario que la información que contiene cada archivo se agrupe con cierto orden. Existen diferentes sistemas de archivos, y estos están "estandarizados" para funcionar según el sistema operativo en el que se vaya a utilizar.

  • NTFS: El sistema de archivos por excelencia de windows
  • EXT: Sistema de archivos desarrollado inicialmente para Linux
  • HFS: Este sistema de archivos para MacOS
  • FAT: Tabla de alocación dde archivos, es un formato populary adimitido por todos los sistemas operativos, se utiliza como sistema de archivos para intercambio, por su alta compatibilidad entre sistemas operativos.

Conclusión

Siempre hay que tener en cuenta la fase de planeación como la más importante en cualquier tipo de trabajo que se haga, hay que tener en cuenta los recursos con los que se cuenta y los que se van a necesitar para realizar dicho trabajo o sistema, ya que ésto nos garantizará el correcto funcionamiento de nuestro sistema
